## What I do

- **Complete business websites.** Production sites for restaurants, clinics and manufacturers, built end to end: design, code, deployment and upkeep.
- **Motion-first frontends.** Scroll-driven storytelling, 3D scenes and micro-interactions. React, TypeScript, Tailwind, Framer Motion, GSAP, Three.js.
- **Creative coding.** Webcam-reactive generative art on canvas: L-systems, procedural drawing, real-time hand tracking with MediaPipe.

## How I work

- Design and code in the same head. Type scale, spacing and easing curves matter to me as much as bundle size and Lighthouse scores.
- I ship. Everything on this profile is live, not just pushed.
- Comfortable owning a project alone, from first mockup to deployed site.
